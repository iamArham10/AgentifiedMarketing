### **PAGE 7: System Configuration**

### **Visual Layout:**

- **Tab-based settings interface:**
    - **Tab 1: Agent Pool Management**
        - List of all agent instances (Agent ID, Type, Status, Trust Score, Uptime)
        - Actions: "Deactivate" (pauses agent), "Remove" (deletes instance), "Clone" (creates duplicate)
        - "Add New Agent" button → Modal with dropdowns (Agent Type, Initial Trust Score, Layer Assignment)
    - **Tab 2: API Integrations**
        - Platform cards: Meta | Twitter | LinkedIn
        - Each card shows:
            - Connection status (green "Connected" badge or red "Disconnected")
            - Last sync timestamp
            - Rate limit usage bars (e.g., "450/500 requests used today")
            - "Reconnect" / "Test Connection" buttons
        - OAuth flow triggers for authorization
    - **Tab 3: Brand Guidelines Repository**
        - File upload area for brand documents (PDFs, images)
        - Uploaded files list with preview thumbnails
        - "Extract Guidelines" button → Triggers NLP agent to parse documents
        - Extracted guidelines displayed as key-value pairs:
            - Tone: "Professional yet approachable"
            - Prohibited words: "Cheap, discount, limited time" (editable)
            - Color palette: HEX codes with color swatches
    - **Tab 4: Trust & Reputation Settings**
        - Sliders:
            - Initial trust score for new agents (default: 50/100)
            - Trust decay rate (penalty per failure)
            - Reputation weight in consensus (0-100%)
        - "Reset All Scores" button (requires confirmation)

### **The Inputs (User Actions):**

- **Agent Pool:** Click "Add New Agent" → Select Type = "Copywriter", Initial Score = 70, Layer = "Creator" → Submit
- **API Integrations:** Click "Reconnect" on Meta card → Redirected to OAuth → Returns with success toast
- **Brand Guidelines:** Drag-drop PDF → "Extract Guidelines" → Review parsed data → Save
- **Trust Settings:** Drag slider to adjust reputation weight → Click "Save Changes"
