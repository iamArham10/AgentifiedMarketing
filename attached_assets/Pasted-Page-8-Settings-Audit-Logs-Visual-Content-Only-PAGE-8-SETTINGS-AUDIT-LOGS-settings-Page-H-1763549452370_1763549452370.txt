USER INTERACTION FLOWS
Flow 1: Change Password

    User on Account tab
    Clicks [Change Password] button
    Modal opens
    User enters current password: "OldPass123!"
    User enters new password: "NewSecurePass456!"
    User confirms new password: "NewSecurePass456!"
    Clicks [Change Password]
    Modal shows loading spinner
    Modal closes
    Toast notification: "Password changed successfully"

Flow 2: Enable 2FA

    User on Security tab
    Sees "Two-Factor Authentication" card with "Disabled" badge
    Clicks [Configure 2FA ‚Üí]
    Modal opens with QR code
    User opens authenticator app on phone
    Scans QR code
    Enters 6-digit code: "123456"
    Clicks [Enable 2FA]
    Modal shows loading
    Modal closes
    Badge updates to "Enabled ‚úì"
    Toast notification: "Two-factor authentication enabled"

Flow 3: Generate API Token

    User on Security tab
    Scrolls to "API Keys & Tokens" card
    Clicks [+ Generate New Token]
    Modal opens
    User types token name: "Production API Access"
    Selects expiration: "Never"
    Clicks [Generate Token]
    Modal shows success with full token: "sk_prod_abc123..."
    User clicks [Copy to Clipboard]
    Toast notification: "Token copied to clipboard"
    User clicks [Done]
    Modal closes
    New token appears in list

Flow 4: Update Preferences

    User on Preferences tab
    User unchecks "Marketing tips" under Email Notifications
    User changes "Default Dashboard View" to "Agent Network"
    User changes "Timezone" to "America/New_York (EST)"
    User checks "Participate in beta features"
    Clicks [Save Preferences]
    Button shows "Saving..." with spinner
    Button shows "Saved ‚úì" briefly
    Toast notification: "Preferences saved successfully"
    Next login will show Agent Network as default dashboard

Flow 5: Filter Audit Logs

    User on Audit Logs tab
    Sees full list of all log entries
    Clicks "All Actions" dropdown
    Selects "Approvals"
    Table filters to show only approval actions (entries 1, 5, 8)
    User types "CM-2847" in search box
    Table further filters to show only entries related to campaign CM-2847 (entries 1, 2)
    User clicks [Export Logs]
    Toast notification: "Generating export..."
    File downloads: "audit_logs_approvals_CM-2847_2025-11-19.csv"
    Toast notification: "Export complete"

Flow 6: View Audit Log Details

    User on Audit Logs tab
    Sees log entry: "Nov 19, 15:52 | sarah@brand | ‚úì Approved Campaign CM-2847"
    Clicks [View Details]
    Detail modal opens
    User sees full information:
        User who approved: Sarah Mitchell
        IP address: 103.255.47.142
        Location: Lahore, Pakistan
        Agent consensus: 3/3 agents, 89.7% confidence
    User expands "View Raw JSON Data"
    Sees full JSON event payload
    User clicks [Copy JSON]
    Toast notification: "JSON copied to clipboard"
    User clicks [Close]
    Modal closes, back to log table

Flow 7: Revoke Session

    User on Security tab
    Sees "Session Management" card with 3 active sessions
    Notices suspicious session: "Firefox on macOS" (1 day ago) but user doesn't use Firefox
    Clicks [Revoke] button next to that session
    Confirmation modal opens: "Revoke Session"
    User reviews device info
    Clicks [Revoke]
    Modal closes
    Session disappears from list with fade animation
    Session count updates: "You are currently signed in on 2 devices"
    Toast notification: "Session revoked successfully"
    New entry appears in Security Logs: "Session revoked for Firefox/macOS"

Flow 8: Delete Account (Full Flow)

    User on Account tab
    Scrolls to Account Actions
    Clicks [Delete Account] (red button)
    Confirmation modal opens with warning
    User reads warning: "This action cannot be undone..."
    Modal requires typing email to confirm
    User types: "sarah@brand.com"
    [Delete Account] button becomes enabled
    User clicks [Delete Account]
    Second confirmation modal: "Are you absolutely sure?"
    User clicks [Yes, Delete My Account]
    Loading overlay appears: "Deleting your account..."
    Account deletion process:
        Delete all campaigns
        Delete all agent data
        Delete all settings
        Revoke all sessions
        Revoke all API tokens
    User automatically logged out
    Redirect to login page
    Message: "Your account has been deleted successfully"
